# Overview

StudySense AI is a modular Python project designed to help students study efficiently by:

Extracting and generating a clean, structured summary from PDF notes.

Creating a personalized quiz (Multiple Choice or True/False) based on the notes.

Calculating scores accurately and displaying results in a friendly, interactive Streamlit interface.

The project combines AI-based PDF parsing with quiz generation, providing a seamless study experience.

# Features

Upload PDF Study Notes: Easily upload your study material in PDF format.

Automated Summary: Extract key points and generate an easy-to-read summary.

Quiz Generation: Automatically create quizzes with multiple-choice or true/false questions.

Interactive UI: Modern Streamlit interface with emojis, icons, and stylish score cards.

Accurate Scoring: Correct answers tracked properly, final score displayed.

Start Over Option: Reset quiz and summary to start a new study session.

Handles Letters + Text: Quiz options with letters (A, B, C, etc.) and text.
